# Softverski sistem za iznajmljivanje bicikala

Ovaj projekat predstavlja klijent-server sistem za upravljanje iznajmljivanjem bicikala u Java okruženju.
Razvijena aplikacija za iznajmljivanje bicikala omogućava registraciju i prijavu prodavca, evidenciju iznajmljivanja i kupaca.

## Struktura projekta

- **softveri_klijent_ANDRIJANA** - Klijentska aplikacija sa grafičkim interfejsom (Java Swing)
- **softveri_server_ANDRIJANA** - Serverska aplikacija koja upravlja zahtevima klijenata
- **softveri_zajednicki_ANDRIJANA** - Zajedničke klase (Operacije, zahtevi i odgovori između klijenata i servera)

## Funkcionalnosti

- **Iznajmljivanje** - ubacivanje novih iznajmljivanja, pretraga i izmena podataka
- **Kupac** - dodavanje, pretraga, izmena i brisanje kupaca
- **Prodavac** - prijava, kreiranje, pretraga, izmena i brisanje prodavca
- **Bicikla** - dodavanje, pretraga, izmena i brisanje bicikala
- **Mesto** - dodavanje, pretraga, izmena i brisanje mesta
- **Termin dežurstva** - ubacivanje, pretraga, izmena i brisanje termina

## Tehnologije

- Java SE
- Java Swing (GUI)
- Socket programming
- MySQL baza podataka
